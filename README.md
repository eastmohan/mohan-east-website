# mohan-east-website

Mohan East's digital portfolio showcasing projects, accomplishments, and other aspects.

## Features

- Static HTML website with responsive design
- Vercel Speed Insights integration for performance monitoring
- LinkedIn and GitHub profile integration

## Speed Insights

This website uses Vercel Speed Insights to track web performance metrics. The integration is configured to automatically load when deployed on Vercel.

### Setup

Speed Insights is already configured in this project. To enable it:

1. Ensure the project is deployed on Vercel
2. Enable Speed Insights in your Vercel project dashboard
3. The `speed-insights.js` script will automatically load and track performance metrics

No additional configuration is required for the Speed Insights integration to work.
