# mohan-east-website

I made a website to show my projects and extracurriculars. I learned a lot about HTML, especially making image carousels, which was significantly more complicated than I think it should have been. It was fun adding content and building my website from scratch. I mostly focus on hardware engineering projects and extracurriculars, so I didn't have a ton of webdev skills going in to this. Most of it I learned while coding the website, which made it take a while at first. However, a lot of the stuff I used many times so I only had to learn it once.

I chose to put some projects on the home page (index.html) because I thought it looked to plain without it. I also added the LinkedIn part and GitHub link to my home page so people might be more likely to click it since they dont have to go to a seperate page. Its important to know that the LinkedIn connect element might not show up if you aren't signed into LinkedIn. There is nothing I can do about this as it is something I embedded using code provided by LinkedIn.
